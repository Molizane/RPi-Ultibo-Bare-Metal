Quick setup — if you’ve done this kind of thing before
HTTPS -> https://github.com/Molizane/Ultibo-16-Key-Keypad.git
or
SSH -> git@github.com:Molizane/Ultibo-16-Key-Keypad.git

We recommend every repository include a README, LICENSE, and .gitignore.


…or create a new repository on the command line

echo "# Ultibo-16-Key-Keypad" >> README.md
git init
git add README.md
git commit -m "first commit"
git remote add origin https://github.com/Molizane/Ultibo-16-Key-Keypad.git
git push -u origin master

…or push an existing repository from the command line

git remote add origin https://github.com/Molizane/Ultibo-16-Key-Keypad.git
git push -u origin master
